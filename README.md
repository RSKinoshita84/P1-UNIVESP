# 🌧️ Sistema de Alerta de Alagamento  
### Projeto Integrador I — Engenharia da Computação — UNIVESP

Este repositório contém o backend do **Sistema de Alerta de Alagamento**, desenvolvido como parte do **Projeto Integrador I** do curso de Engenharia da Computação da UNIVESP.  
O objetivo do sistema é **monitorar condições climáticas e ambientais**, identificar riscos de alagamento e **emitir alertas automáticos** para usuários cadastrados.

---

## 🚀 Tecnologias Utilizadas

- **Python 3.10+**
- **FastAPI** — Framework para APIs rápidas e modernas
- **Uvicorn** — Servidor ASGI de alta performance
- **SQLAlchemy** — ORM para manipulação do banco de dados
- **SQLite / PostgreSQL** (dependendo da configuração)
- **Pydantic** — Validação de dados
- **Requests** — Consumo de APIs externas
- **Jinja2** — Templates para notificações
